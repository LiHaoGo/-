## 使用原生js实现的类似全民飞机大战的小游戏(优化中)

##### 项目描述：通过css和js实现的一个小游戏，在开始页面可以选择困难程度（难度的等级是采取了子弹发射的速度和敌机的血条来决定的），然后通过js动态创建敌机和本机，在运行过程中会根据击落的敌机不同，累加得分，最后会显示最终得分。

##### 技术选型：HTML5 + CSS3 +Javascritp

##### 运行方式：clone此项目，然后双击打开index.html
